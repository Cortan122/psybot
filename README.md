# Psychological Essence.
### Telegram bot.

## Starting the bot

```console
$ echo 'API_TOKEN=something' >> .env
$ echo 'DATABASE_URI=something' >> .env
$ ./run.sh
```

## Install pre-commit hooks

```console
$ pip install pre-commit
$ pre-commit install
```
